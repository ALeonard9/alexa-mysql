# alexa-mysql
An AWS Alexa Skill that integrates with a MYSQL backend for the betting app. Integrates with personal betting application on https://www.aleonard.us.

## Technology used
* Node.js
* AWS Alexa-skillset
* AWS RDS Mysql
* AWS Lambda
